## Research project scraper


Information from https://biowikifarm.net/v-mfn/forschungsprojekte/Hauptseite

### Usage

BiowikifarmWebScraperCommand 
```
php BiowikifarmScraperCommand.php username password filename
```

BiowikifarmAnnotatorCommand
```
php BiowikifarmAnnotateCommand.php filename 
```

BiowikifarmStatisticCommand
```
php BiowikifarmStatisticCommand.php filename 
```
